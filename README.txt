Ce micro blog vous permet d'afficher des commentaires, postés par divers utilisateurs qui doivent passer
par une procédure d'inscription, ou de connexion.

Les droits des utilisateurs :
- Le micro blog est de type collaboratif, chaque membre peut modifier ou supprimer un message posté
- Chaque utilisateur peut poster un message quand il le souhaite

Le micro blog affiche par page les commentaires, cinq à la fois.

Une fonction de recherche est mise à disposition, par mot clé, qui se chargera de référencer les messages concernés.
